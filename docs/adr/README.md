# Architecture Decision Records

文件命名：`NNNN-<kebab-title>.md`（NNNN 4 位顺序号）。

## 模板

```
# NNNN: 决策标题

- 状态：Proposed | Accepted | Superseded by NNNN | Deprecated
- 日期：YYYY-MM
- 关联 Phase / todo：

## 背景

## 选项
- 方案 A：…
- 方案 B：…

## 决定
…

## 影响
…
```
